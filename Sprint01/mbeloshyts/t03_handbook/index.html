HTML

Introduction to HTML

HTML is the standard markup language for creating web pages.

<!DOCTYPE html>

Example:

<html>
  <head>
    <title>Title</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>

HTML Elements

HTML documents are composed of HTML elements represented by tags.

<p>This is a paragraph</p>

Example:

<div>
  <h2>Heading</h2>
  <p>Paragraph</p>
</div>

HTML Attributes

HTML attributes provide additional information about HTML elements.

<img src="image.jpg" alt="Image">

Example:

<a href="https://www.example.com">Link</a>

HTML Forms

HTML forms allow users to input data that can be submitted to a server for processing.

<form action="/submit" method="post">

Example:

<input type="text" name="username">

CSS

Introduction to CSS

CSS is a style sheet language used for describing the presentation of a document written in HTML.

body { background-color: lightblue; }

Example:

body {
  font-family: Arial, sans-serif;
}

CSS Selectors

CSS selectors are patterns used to select and style elements in an HTML document.

p { color: blue; }

Example:

#header {
  font-size: 24px;
}

CSS Box Model

The CSS box model describes the space around an element, including padding, borders, and margins.

.box { padding: 10px; }

Example:

.box {
  border: 1px solid black;
}

CSS Flexbox

CSS Flexbox is a layout model that allows for more efficient and predictable positioning of elements within a container.

.container { display: flex; }

Example:

.item {
  flex-grow: 1;
}
